# AjaxSPA

*Classe para tornar páginas html/PHP para o padrão SPA*

## Requisitos
	- O site deve está em um servidor com apache
	- Arquivo .htaccess deve está configurado
	- Lib Jquery.

## Modo de uso
- Na pagina index
	- importe a lib jquery normamente
	- importe a class AjaxSpa
- Abra uma TAG script
